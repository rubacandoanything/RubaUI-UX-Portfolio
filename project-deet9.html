Sankhyana Website Dashboard

UI/UX Design & Flow Architecture

Contributed to the internal dashboard for managing the Sankhyana website, working specifically on the Blog Management, News Management, and User Management modules. The dashboard allows admins to add, edit, and delete blog posts, categorize content, manage visibility, and assign tags. In the News Management section, admins can schedule news updates, manage sources, and track status. The User Management section enables admins to view user profiles, manage access, and monitor activity. The design emphasizes usability, clean navigation, and content clarity to ensure smooth backend operations.

Release Date 2024.04
Services Provided UI/UX Design in Figma · Flow Mapping for Admin Tasks · Blog Management UI · News Scheduling & Status Module · Role-Based User Management · Dashboard Structure Planning · Table & Card UI Layouts · Admin Panel Usability Design
